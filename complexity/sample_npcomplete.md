# NP-Completeness: Example note

Canonical NP-Complete problem: SAT (Boolean satisfiability)

Overview:
- SAT asks whether a given Boolean formula (in CNF) has a satisfying assignment.
- 3-SAT is a restricted version where each clause has exactly 3 literals; 3-SAT is NP-Complete.

Notes on reductions:
- To show a problem is NP-Complete, demonstrate it is in NP and provide a polynomial-time reduction from a known NP-Complete problem (often 3-SAT).
- Include worked examples in this folder to illustrate common reduction techniques.
