# Theory of Computation

## Finite Automata: 
DFAs, NFAs, and Regular Expressions (Regex). 
## Context-Free Languages: 
Pushdown Automata (PDA) and Context-Free Grammars (CFG).  

## Turing Machines (TM): 
Decidability, Halting Problem, and the Church-Turing Thesis. 
## Reducibility: 
Proving languages are undecidable.  

## Complexity Classes: 
$P$ vs $NP$, $NP$-Complete, and $NP$-Hard problems.
