# Theory of Computation

A concise, well-organized reference and learning repository covering core topics in theoretical computer science: finite automata, context-free languages, Turing machines and decidability, reducibility, and complexity theory (including P vs NP and NP-completeness).

## About

This repository contains notes, examples, and exercises for the Theory of Computation. It is intended as a study aid for students and practitioners who want a practical, example-driven introduction to the subject.

## Topics Covered

- Finite Automata
  - Deterministic Finite Automata (DFA)
  - Nondeterministic Finite Automata (NFA)
  - Regular expressions and conversions between DFAs/NFAs/regex
- Context-Free Languages
  - Pushdown Automata (PDA)
  - Context-Free Grammars (CFG) and grammar transformations
  - Parse trees and derivations
- Turing Machines
  - Formal definition of Turing machines
  - Decidability, recognizability, and the Halting Problem
  - Church–Turing thesis and its implications
- Reducibility and Undecidability
  - Reductions between decision problems
  - Techniques to prove languages are undecidable
- Complexity Theory
  - Complexity classes: P, NP, NP-Complete, NP-Hard
  - Reductions for NP-Completeness

## Getting Started

1. Clone the repository:

   git clone https://github.com/Venkatesh123-start/theory_of_computation.git

2. Browse the folders and files to find notes, automaton diagrams, example Turing machines, and exercise sets.

3. Use the examples as templates to build and test your own automata and grammars.

## Examples & Exercises

This repository aims to include worked examples for each topic and exercises with suggested solutions. Look for directories and files named after the topic (for example, `finite_automata/`, `pushdown_automata/`, `turing_machines/`, `complexity/`) to find examples.

## Recommended Resources

- "Introduction to the Theory of Computation" — Michael Sipser
- "Automata Theory, Languages, and Computation" — Hopcroft, Motwani, and Ullman
- Lecture notes and problem sets from university courses in theoretical computer science
